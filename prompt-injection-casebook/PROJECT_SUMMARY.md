# Project Summary

## What This Is

Prompt Injection Casebook is a structured documentation project about LLM security and AI safety failure modes.

## Why It Exists

Modern AI systems increasingly read documents, browse websites, summarize emails, use tools, and automate decisions. Prompt injection becomes risky when untrusted content can influence model behavior or tool use.

## What It Demonstrates

This repository demonstrates:

- AI security documentation
- LLM safety awareness
- structured technical writing
- failure mode analysis
- defensive recommendations
- evaluation planning
- responsible communication of dual-use topics

## How It Supports an AI Safety Portfolio

This project can be linked in a CV or application as a technical writing and security documentation sample related to LLM evaluation and prompt-injection risk.
