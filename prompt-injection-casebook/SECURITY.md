# Security Policy

## Purpose

This repository is intended for defensive AI safety education, documentation, and evaluation planning.

## Responsible Use

Do not use this project to:

- attack real systems
- bypass access controls
- extract private data
- steal credentials
- automate harmful activity
- publish exploit instructions against live targets

## Reporting Issues

If you find a problem in this repository, such as unsafe wording, unclear mitigation guidance, or accidental inclusion of sensitive information, please open an issue or submit a pull request.

## Safety Standard

Examples should remain sanitized, high-level, and focused on defensive learning.
